# chat-app
Chat Application (MERN Stack + Socket.io)

npm packages used: 
1. bcryptjs - for encryption of password
2. cloudinary
3. cookie-parser
4. dotenv
5. express
6. jsonwebtoken - generating of JWT
7. mongoose - to connect to MongoDB
8. socket.io
9. nodemon - dev testing

Frontend:
1. React + Vite
2. Tailwind CSS
3. Zustand
4. Daisy UI

Backend / API:
1. Node.js
2. MongoDB
3. Express
3. Socket.io
